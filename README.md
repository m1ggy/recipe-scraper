# Recipe Scraper

Recipe Scraper is a node application that scrapes any website that uses [h-recipe](https://microformats.org/wiki/h-recipe) format.

## Development

To start developing run the following commands

1. Install Node via NVM

   ```bash
   nvm use
   ```

2. Install packages

   ```bash
   npm install
   //or
   pnpm install
   //or
   yarn
   ```

3. run app

   ```bash
   npm run dev
   //or
   pnpm dev
   //or
   yarn dev
   ```
